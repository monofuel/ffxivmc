Welcome!

The goal of this site is to catalog market orders from the server Lamia and figure out what the most profitable items to gather and craft are. This website is still a work in progress, so expect some rough edges.

The most useful component of this website is our ffxivapp plugin, which can be found Here and will forward any market data it finds to this website. The plugin is also open source on github.
